## Plan para Deployment a Vercel

El proyecto ya tiene la configuración necesaria:
- `vercel.json` con rewrites para SPA
- Script `npm run deploy:vercel` en package.json

**Pasos para ejecutar:**

1. Ejecutar `npm run deploy:vercel` en la terminal
2. Seguir las instrucciones de Vercel CLI (si es primera vez, pedira login)
3. Confirmar el deployment a producción

**Nota:** El proyecto usa Vite + React, y ya está configurado correctamente para Vercel.
